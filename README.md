# Firefly Synchronization Simulation

This project simulates the emergent synchronization behavior of fireflies based on their biological clock dynamics and mutual influence through neighbor interactions. The model is inspired by natural phenomena such as synchronized flashing observed in real firefly swarms, where individual fireflies adjust their internal clocks in response to nearby flashes, leading to collective order without a central coordinator. I implemented this simulation using Python with NumPy for numerical computations, Matplotlib for visualization, and ImageIO for creating animations.

## Model Implementation

I built the simulation around a network of fireflies arranged on an N x M grid, where each firefly maintains an internal phase value between 0 and 1 representing its biological clock. When a firefly's phase reaches 1, it flashes, resets its own phase to 0, and adds a coupling value to the phases of all neighboring fireflies within a specified radius r. I implemented periodic boundary conditions to create a toroidal grid, preventing edge effects from artificially limiting interactions. The simulation progresses in discrete time steps, with each firefly's phase increasing by a fixed increment dt at every step, where dt equals 1 divided by timestepsPerPeriod. This approach ensures that an isolated firefly would complete one full cycle in exactly timestepsPerPeriod steps.

## Order Parameter Definition

To quantify the degree of synchronization across the swarm, I defined an order parameter based on the circular representation of phases. I mapped each firefly's phase value to a point on the unit circle using complex exponentials, calculating exp(2πj × phase) for every firefly. The order parameter R(t) is then the magnitude of the average of these complex vectors across all fireflies. This value ranges from 0 in a completely disordered state where phases are uniformly random to 1 in a perfectly synchronized state where all fireflies have identical phases. This definition effectively captures collective synchronization because the vectors cancel when phases are scattered and reinforce when they align.

## Effect of Neighborhood Radius

I investigated how the neighborhood radius r affects synchronization by running simulations with r values of 0.5, 1.5, and 3.0 while keeping the coupling value constant at 0.08 on a 20x20 grid. For each radius, I initialized all fireflies with random phases and evolved the system for 15 periods with 100 time steps per period. With r = 0.5, representing very local interactions where fireflies only influence their immediate neighbors, I observed slow synchronization and a lower final order parameter value, as the limited interaction range prevented information from propagating efficiently across the grid. For r = 1.5, a moderate neighborhood radius, the system achieved faster and stronger synchronization, reaching a high order parameter value within the simulation time. With r = 3.0, where each firefly interacts with a large portion of the grid, I observed almost instantaneous synchronization, with the order parameter rising rapidly to near 1. These results demonstrate that larger neighborhood radii significantly accelerate synchronization by enabling each flash to influence more neighbors, creating stronger and faster information propagation throughout the network.

## Effect of Coupling Value

I examined the impact of different coupling strengths by varying the coupling value from 0.02 to 0.2 while keeping the neighborhood radius fixed at r = 1.5 on a 20x20 grid. With a low coupling value of 0.02, representing weak interactions between fireflies, I observed very slow synchronization with the order parameter rising only gradually over time and never reaching a high value within the simulation period. For a moderate coupling value of 0.08, the system achieved faster synchronization with a steady increase in the order parameter toward values close to 1. With a high coupling value of 0.2, I observed very rapid synchronization where the order parameter approached 1 within the first few hundred time steps. These results indicate that stronger coupling accelerates synchronization, but I noted that excessively high coupling values could potentially lead to phase overshoot effects where fireflies advance too quickly through their cycles, though within the tested range the behavior remained stable.

## Effect of Grid Size

I explored how the overall network dimensions affect synchronization by simulating grids of sizes 10x10, 20x20, and 40x40 while keeping the neighborhood radius at r = 1.5 and coupling value at 0.08. For the small 10x10 grid, synchronization occurred rapidly with the order parameter rising quickly to a high value, as the limited number of fireflies allowed information to propagate across the entire system in fewer steps. The medium 20x20 grid showed slower synchronization, taking more time to reach comparable order parameter values due to the increased distance between far-apart fireflies. For the large 40x40 grid, I observed significantly slower initial synchronization, with the order parameter rising much more gradually and requiring additional time to approach high values. These results demonstrate that grid size substantially affects synchronization dynamics, with larger networks requiring more time for cascades to propagate globally and achieve collective order.

## Visualization and Animation

I created an animation to visually demonstrate the emergence of synchronization across the firefly network over time. Using a 20x20 grid with r = 1.5 and coupling value 0.08, I simulated 15 periods and saved the phase states at each time step. I then generated a GIF animation showing the spatial evolution of phases, using a color map where colors represent the current phase value of each firefly. The animation clearly illustrates how initially random phases gradually organize into synchronized patterns, with patches of synchronized fireflies forming and eventually merging until the entire grid flashes in unison. This visual representation effectively complements the quantitative order parameter analysis by showing the spatial patterns of synchronization emergence.

## Conclusions

Through this simulation study, I demonstrated that firefly synchronization emerges from simple local interactions without any central coordination. The neighborhood radius, coupling strength, and network size all play crucial roles in determining synchronization speed and quality. Larger interaction ranges and stronger coupling both accelerate synchronization, while larger networks require more time to achieve global order. The order parameter based on circular phase representation provides an effective quantitative measure of synchronization, and the spatial animation reveals the beautiful patterns that emerge as order propagates through the system. These findings align with observations of natural firefly swarms and contribute to understanding self-organization phenomena in complex systems.
